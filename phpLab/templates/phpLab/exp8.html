Theory


1.1 Introduction

A function provides a convenient way to encapsulate computations. It helps to perform some task repetitively at different points in the program. The function is a block of code written to perform a particular task. Functions are of two types viz. built-in functions and user defined functions.

1.2 Built-in function

There is no need to write the code for built-in functions. User just calls the function based on the requirement. PHP provides many built-in functions. Some of them are count(), sort(), date() etc.

1.3 User defined functions

User can define (write code for) his/her own functions based on the requirement.
The syntax for defining function is:
function function_name(argument list)
{
body of the function;
}
The function can be called by using the function name along with the parameter(s)/argument(s). The function returns value using return keyword.
A function can have any number of arguments and any type that is supported by PHP. The Argument may have default value.
