Recursive Function

Theory

1.1 Introduction

A function can call another function or the function itself. If the function calls itself it is called Recursive function . To avoid infinite looping, while writing recursive functions one needs to add an end condition.

function calc($var)
{
….
}
function fun($arg)
{ 


if (small($arg))        //end condition
  return calc($arg); 
else
…..
  return fun($arg1);    //recursive call
}
In the above example function calc() is iterative and function fun() is recursive.For recursive functions, equivalent iterative function can be written and vice versa. For every recursive call, separate memory allocation is required to switch over to next recursive call. This is not needed in its iterative implementation. But writing functions using recursion is more easier.

1.2 Example

To display a number of terms in the fibonacci series. Let $num be the index of the fibonacci number in the series

function fibo($num)
{ 


if ($num==1 || $num==2)
    return $num;
else
    return fibo($ num-1) + fibo($num-2));
}
