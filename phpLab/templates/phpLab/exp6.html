Array

Theory


1.1 Introduction
We learnt from previous experiments that a variable can hold only one value at a time. To store more than one value of same data type with single variable name , we use an array. Array is a collection of key value pairs.
In PHP an array is created by using array() function.
e.g. $arr = array();
This creates an empty array. Three elements can be stored separately in the array as shown below:
$arr[0]=10;
$arr[1]=20;
$arr[2]=30;
Alternatively,the above array can be created by using function array() as shown below:
$arr = array(10,20,30);
PHP supports array with numeric index values and the index starts with a value 0. An array element can be accessed by using array_name[key] syntax. e.g. $arr[1]=20;
here '1' is the index value that refers to the element 20. 1.2 Multidimensional arrays
Each element of the multidimensional array is another array. To store marks and roll numbers of two students, the array can be defined as shown below: $marks=array( array(1,67,89,78), array(2,78,90,77) ); PHP provides variety of built-in functions related to Array. Some of the Primitive operations on array provided by PHP are mentioned below:
count() returns the number of elements stored in an array.
sort() sorts array in ascending order.
rsort() sorts array in descending order.
array_search() searches the array for given key and returns the key value if successful.
array_reverse() returns the array elements in reverse order.
