While Loop

Theory

1.1 Introduction
PHP provides looping constructs which are helpful in executing one or more statements repetitively till some condition is satisfied. In short for repetitive execution of block of code, these constructs can be used. PHP provides looping constructs like while, do-while, for , for each .
The basic format of while statement is :
while(condition)
{
one or more statements to be executed;
}
Here the condition is checked first. If it is true(non zero) then the statement(s) inside the loop is/are executed. This execution is repeated till the condition is true.
Like while, the for loop is another entry-controlled looping construct. The basic syntax for statement is for( initialization; test condition; updation )
{
one or more statements to be executed;
}
is equivalent to
initialization;
while(test_condition)
{
One or more statements ;
Updation;
}
Initialization part initializes the counter variables, the condition to be checked is written in the test condition part. If condition is true(non-zero), the code is executed otherwise not. The updation of the counter is done in the last part. Initialization takes place only once at the start of the loop.

1.2 Example of ‘while’
To calculate the sum of n numbers stored in an array :
$num = 10;
$sum = 0;
