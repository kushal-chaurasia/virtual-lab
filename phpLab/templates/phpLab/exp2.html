If Statement

Theory


1.1 Introduction

In the PHP script, one can change the order of sequence of execution. Like other programming languages, PHP also provides Control Flow Structures or Statements. PHP provides Conditional Flow Control statements such as if, if...else , if...else ..if and switch statement.


1.2 If Statement

The syntax for if statements is:
if(test_expression )
{
One or more statements;
}

If the expression is true then one or more statements in if block will be executed.Further, if statement can be nested i.e. we can put if statement within another if statement.


1.3 If... Else Statement

The syntax for if... else statement is:
if(test_expression )
{
One or more statements; }
else
{ One or more statements; }

If the expression is true then one or more statements written within the if block will be executed otherwise those of else block will be executed. If multiple conditions are present then this can be further extended as:
if(test_expression )
{ One or more statements;
}
else if(expression){
One or more statements; } else{
One or more statements;
}
