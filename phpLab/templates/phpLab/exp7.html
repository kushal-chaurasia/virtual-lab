Array Associative

Theory


1.1 Introduction

Associative array is also called as an array with keys.The syntax for creating an array is
array(
key1 => value1,
key2 => value2,
key3 => value3,
…);
The key can be string.
e.g. $student= array("SE" =>120,"TE" => 125 ,"BE" =>100);
Here string is used as an index of the array. The classes (SE, TE, BE) of students are associated with the number of students of each class in the college.
For example, to print the number of students in class “TE”, the following statement will be used:
echo $student["TE"] ;
This will print 125 .

1.2 For Each Loop

To print the values and scan through the associative array foreach loop can be used.
The basic syntax of foreach loop is :
foreach (array_expression as $value)
statement;
Or
foreach (array_expression as $ key=>$ value)
statement;
For example, to print the values of the array "student" (created above), foreach statement can be used as shown below:
foreach ($student as $s => $s_value)
echo $ s." ".$ s_value ."<br>";
